# Insult Sword Fighting

Open-source browser game inspired by Monkey Island insult duels. Player vs AI NPCs, LLM-judged.

**Status:** v1 MVP in development.

## Stack

SvelteKit 5 + Cloudflare (Workers, D1, KV, Vectorize) + Drizzle + Better Auth + Resend + Workers AI.

## Local development

```bash
pnpm install
pnpm dev
```

See `DESIGN.md` for architecture and `docs/superpowers/specs/` for the v1 design spec.
